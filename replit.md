# Amna Ghafoor - Personal Portfolio Website

## Overview
A minimal, modern personal portfolio website for Amna Ghafoor, inspired by Ali Abdaal's aesthetic. Built with pure HTML, CSS, and JavaScript featuring a soft pink and purple color scheme.

## Project Purpose
Showcase Amna Ghafoor's work as a Data Scientist & AI Enthusiast, featuring her projects, skills, and professional background.

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Google Fonts (Inter)
- **Design**: Minimal, responsive design with smooth animations

## Color Palette
- Base Background: `#FFFFFF` (White)
- Light Pink Accent: `#FFF0F5`
- Light Purple Accent: `#D8B4F8`
- Dark Purple Text: `#4B0082`

## Sections
1. **Navigation**: Sticky navbar with smooth scrolling
2. **Hero**: Name, title, bio, and action buttons
3. **About**: Personal background and experience
4. **Projects**: 9 project cards with hover effects
5. **Skills**: Grid layout showcasing technical and soft skills
6. **Contact**: Contact information and form
7. **Footer**: Copyright notice

## Features
- Fully responsive design (mobile + desktop)
- Smooth scroll animations
- Fade-in effects on scroll
- Interactive project cards with hover effects
- Mobile-friendly hamburger menu
- Contact form with validation
